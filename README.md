# Walker Page Prototype v1

This is the first design iteration of what a walker's booking page looks like.

**Design goals:**
- Warm, personal, trustworthy
- Shows walker photo, bio, service area
- Clear pricing with service descriptions
- Payment methods visible (Venmo, Zelle, Cash, Apple Pay)
- Booking request form that sends email notification
- Feels like the walker owns this — not a generic app profile

**Color scheme:** Forest green (#1B4332) + cream (#FDFAF5)
**Font:** Outfit
**Tone:** Direct, first person, no corporate filler
